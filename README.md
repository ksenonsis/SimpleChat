# SimpleChat — простой TCP-мессенджер

## Как запустить

1. Открой `SimpleChat.sln` в **Visual Studio 2022**
2. Запусти проект `SimpleChat.Server` (консольное окно)
3. Запусти `SimpleChat.Client` — можно несколько раз
4. В клиенте: введи имя → нажми **Подключиться**
5. Пиши сообщения, они видны всем!

---

## Что делает каждый файл

### Сервер (`SimpleChat.Server/Program.cs`)
- Ждёт подключений на порту **5000**
- Каждый новый клиент обрабатывается в отдельном **потоке (Thread)**
- Получает сообщения и рассылает их всем остальным
- Ведёт список подключённых пользователей

### Клиент (`SimpleChat.Client/`)
- `MainWindow.xaml` — внешний вид окна
- `MainWindow.xaml.cs` — весь код (нажатия кнопок, получение сообщений)

---

## Как работает протокол (очень просто!)

Клиент и сервер общаются обычными **строками** через `StreamReader` / `StreamWriter`.

```
Клиент подключился → отправляет: "Аня"
Сервер отвечает всем: ">>> Аня вошла в чат."
Сервер отправляет список: "USERS:Аня,Боря"

Аня пишет: "Привет!"
Сервер рассылает всем (кроме Ани): "Аня: Привет!"
```

Если сообщение начинается с `USERS:` — это обновление списка пользователей.
Всё остальное — обычный текст для чата.
